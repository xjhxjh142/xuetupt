# 概率与统计

## 概率基础
- **定义**：事件发生的可能性大小，P(A) = 事件A的结果数 / 总结果数
- **范围**：0 ≤ P(A) ≤ 1
- **必然事件**：P=1，**不可能事件**：P=0
- **引导线索**：
  - 引导：抛一枚硬币正面朝上的概率是多少？抛两次都正面呢？
  - 引导：盒子里3红2白球，摸到红球的概率？
  - 引导：为什么概率总是在0和1之间？
- **常见误解**：误以为概率为0的事件一定不会发生（连续型分布中可能发生）

## 树状图与列表法
- **用途**：列举所有等可能结果
- **适用**：两步或三步随机试验
- **引导线索**：
  - 引导：抛两次硬币，所有可能的结果怎么列出来不遗漏？
  - 引导：树状图和列表法哪个更清晰？什么时候用哪个？
- **常见误解**：画树状图时分叉数漏掉

## 频率与概率
- **频率**：试验中事件发生的次数/总试验次数
- **关系**：大量重复试验时，频率稳定于概率
- **引导线索**：
  - 引导：抛10次硬币得到7次正面，正面的概率是0.7吗？
  - 引导：抛1000次呢？会接近多少？
- **常见误解**：误以为频率等于概率（频率只是概率的估计）

## 统计量
- **平均数**：数据和÷个数，易受极端值影响
- **中位数**：排序后中间的数，不易受极端值影响
- **众数**：出现次数最多的数
- **方差**：衡量数据波动大小，方差越大波动越大
- **引导线索**：
  - 引导：班上有人考100有人考0，用平均数还是中位数代表整体水平？
  - 引导：两个班的平均分一样，但一个方差大一个方差小，说明了什么？
- **常见误解**：只关注平均数而忽视数据的分布

## 统计图
- **条形图**：比较各类别数量
- **扇形图**：显示各部分占比
- **折线图**：展示数据变化趋势
- **直方图**：显示数据分布
- **引导线索**：
  - 引导：想展示班级各科成绩的对比，用什么图？
  - 引导：想展示一个同学一学期的成绩变化趋势，用什么图？
- **常见误解**：直方图（连续分组）和条形图（分类）的混淆
