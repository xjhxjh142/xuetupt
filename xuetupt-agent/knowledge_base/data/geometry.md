# 几何

## 三角形内角和
- **定义**：三角形三个内角的和等于 180°
- **证明**：过顶点作对边的平行线，利用平行线性质证明
- **引导线索**：
  - 引导：想想一条直线是多少度？
  - 引导：如果过三角形一个顶点作对边的平行线，你能发现什么角相等？
  - 引导：这三条平行线创造了哪些同位角和内错角？
- **常见误解**：误以为内角和总是 180°，但这是在平面几何中成立，在曲面几何中不成立

## 勾股定理
- **定义**：在直角三角形中，两条直角边的平方和等于斜边的平方，a² + b² = c²
- **证明**：
  - 面积法证明：用四个全等的直角三角形拼成正方形，通过面积关系推导
  - 赵爽弦图：中国古代的证明方法
- **引导线索**：
  - 引导：一个直角边为3和4的直角三角形，你能用四个这样的三角形拼成一个正方形吗？
  - 引导：拼成的大正方形边长是多少？面积呢？
  - 引导：四个三角形的总面积是多少？中间空白正方形的面积呢？
  - 引导：你能写出它们之间的面积关系吗？
- **常见误解**：误以为勾股定理对所有三角形成立（实际只对直角三角形成立）

## 全等三角形
- **定义**：两个三角形形状和大小完全相同，可以完全重合
- **判定定理**：
  - SSS：三边对应相等
  - SAS：两边及其夹角对应相等
  - ASA：两角及其夹边对应相等
  - AAS：两角及其中一角的对边对应相等
  - HL：直角三角形斜边和一条直角边对应相等
- **引导线索**：
  - 引导：要确定一个三角形的形状，最少需要知道几个条件？
  - 引导：如果知道两条边，能确定唯一的三角形吗？试试看
  - 引导：两条边和它们的夹角呢？
- **常见误解**：SSA（两边及非夹角）不能判定全等

## 相似三角形
- **定义**：对应角相等、对应边成比例的三角形
- **判定定理**：AA、SAS（边成比例）、SSS（边成比例）
- **引导线索**：
  - 引导：全等和相似有什么联系和区别？
  - 引导：如果两个三角形的角都对应相等，边有什么关系？
  - 引导：你能从生活中找到相似三角形的例子吗？
- **常见误解**：误以为全等是相似的特殊情况（正确，全等是相似比为1的特殊相似）

## 圆
- **定义**：到定点距离等于定长的所有点的集合
- **相关概念**：圆心、半径、直径、弦、弧、扇形、圆周角、圆心角
- **圆周角定理**：同弧所对的圆周角是圆心角的一半
- **引导线索**：
  - 引导：自行车轮子为什么是圆的而不是方的？
  - 引导：圆上固定两点，在圆上移动第三点，你发现这个角变了吗？
  - 引导：直径所对的圆周角是多少度？为什么？
- **常见误解**：误以为圆的半径和直径没有区别

## 平行四边形
- **定义**：两组对边分别平行的四边形
- **性质**：对边平行且相等、对角相等、对角线互相平分
- **特殊类型**：矩形（角为直角）、菱形（边相等）、正方形（兼具两者）
- **引导线索**：
  - 引导：长方形拉歪了变成什么？哪些性质变了哪些没变？
  - 引导：对角线把平行四边形分成两个什么图形？
  - 引导：如何判断一个四边形是平行四边形？
- **常见误解**：误以为对角线相等的四边形一定是矩形（需要先判定是平行四边形）
