# 数论基础

## 有理数
- **定义**：可以表示为 p/q（q≠0，p、q为整数）的数
- **分类**：
  - 整数：正整数、0、负整数
  - 分数：有限小数、无限循环小数
- **引导线索**：
  - 引导：1/3 写成小数是什么？它是有限小数吗？
  - 引导：有理数在数轴上都能找到对应点吗？反过来呢？
- **常见误解**：误认为无限小数都是无理数（循环小数是有理数）

## 无理数
- **定义**：不能表示为 p/q 的实数，无限不循环小数
- **常见例子**：√2、√3、π、e
- **引导线索**：
  - 引导：√2 是无限不循环小数，怎么证明的？
  - 引导：边长为1的正方形对角线长度是有理数吗？
- **常见误解**：误以为π = 22/7，实际上22/7只是近似值

## 实数
- **定义**：有理数和无理数的总称
- **实数轴**：每个实数对应数轴上唯一的点
- **引导线索**：
  - 引导：数轴上的每个点都对应一个实数吗？
  - 引导：有理数和无理数哪个多？
- **常见误解**：误以为无理数比有理数少（实际上无理数更多，不可数无穷）

## 绝对值的几何意义
- **定义**：数轴上点到原点的距离，|a| = a(a≥0) 或 -a(a<0)
- **性质**：|a| ≥ 0，|ab| = |a||b|，|a+b| ≤ |a| + |b|（三角不等式）
- **引导线索**：
  - 引导：|x| = 3 在数轴上表示什么？x 有几个可能的值？
  - 引导：|x-1| = 3 表示什么意思？x 到 1 的距离是 3
- **常见误解**：误以为|-a| = a（正确，但需理解 a 是负数时的情况）

## 幂与科学记数法
- **定义**：a^n 表示 n 个 a 相乘
- **科学记数法**：将一个数写成 a × 10^n 形式，其中 1 ≤ |a| < 10
- **引导线索**：
  - 引导：10³ = 1000，10⁰ = 1，那 10⁻¹ = ?
  - 引导：0.000123 用科学记数法怎么表示？
- **常见误解**：误以为 a 可以小于 1 或大于等于 10

## 绝对值不等式
- **|x| < a** 的解集：-a < x < a
- **|x| > a** 的解集：x < -a 或 x > a
- **引导线索**：
  - 引导：|x| < 3 在数轴上表示什么？距离原点小于3的点的集合
  - 引导：|x-2| < 1 表示什么意思？
- **常见误解**：解 |x| > a 时忘记取并集
