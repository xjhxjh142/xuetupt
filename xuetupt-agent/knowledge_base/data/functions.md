# 函数

## 函数的概念
- **定义**：在一个变化过程中，有两个变量x和y，对于x的每一个确定的值，y都有唯一确定的值与之对应
- **表示方法**：解析式法、列表法、图像法
- **引导线索**：
  - 引导：自动售货机投入不同硬币，为什么出的饮料是确定的？
  - 引导：一个x对应多个y还是函数吗？为什么？
- **常见误解**：误以为一个y可以对应多个x就不是函数（这是允许的）

## 正比例函数
- **定义**：y = kx (k ≠ 0)
- **性质**：图像过原点，k > 0 经过一三象限，k < 0 经过二四象限
- **引导线索**：
  - 引导：k=1和k=2的图像有什么不同？
  - 引导：正比例函数和比例有什么关系？
- **常见误解**：误以为y=kx一定过原点是任意的

## 反比例函数
- **定义**：y = k/x (k ≠ 0)
- **性质**：图像是双曲线，k > 0 在一三象限，k < 0 在二四象限
- **引导线索**：
  - 引导：矩形面积固定，长和宽是什么关系？
  - 引导：x越大y怎么变化？x趋近0时y会怎样？
- **常见误解**：误以为反比例函数图像的两个分支会相连（实际不会）

## 变量与常量
- **常量**：保持不变的值
- **变量**：可以变化的值
- **自变量**：主动变化的量
- **因变量**：随自变量变化的量
- **引导线索**：
  - 引导：你在走路时，时间和路程哪个是自变量？
  - 引导：速度不变时，路程和时间是什么关系？

## 函数图像
- **意义**：直观反映变量之间的关系
- **读图**：横轴自变量、纵轴因变量
- **引导线索**：
  - 引导：从图像上怎么看出函数是递增还是递减？
  - 引导：两条函数图像相交的点表示什么？
- **常见误解**：读图时混淆横纵坐标的意义
