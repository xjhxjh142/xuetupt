# 代数

## 一元一次方程
- **定义**：含有一个未知数，且未知数的最高次数为1的方程，形如 ax + b = 0 (a ≠ 0)
- **解法**：移项 → 合并同类项 → 系数化为1
- **引导线索**：
  - 引导：天平的左边放一个苹果加2个砝码，右边放5个砝码，苹果多重？
  - 引导：等式的两边同时加减同一个数，等式还成立吗？
- **常见误解**：移项忘记变号

## 二元一次方程组
- **定义**：含有两个未知数的一次方程组成的方程组
- **解法**：代入消元法、加减消元法
- **引导线索**：
  - 引导：一个未知数不知道时我们设它为x，两个都不知道怎么办？
  - 引导：能先用一个未知数表示另一个吗？
- **常见误解**：代入时漏括号导致符号错误

## 一元二次方程
- **定义**：形如 ax² + bx + c = 0 (a ≠ 0)
- **解法**：因式分解法、公式法、配方法
  - 求根公式：x = [-b ± √(b² - 4ac)] / (2a)
- **判别式**：Δ = b² - 4ac
  - Δ > 0：两个不等实根
  - Δ = 0：两个相等实根
  - Δ < 0：无实根
- **引导线索**：
  - 引导：x² = 4 怎么解？开平方为什么会得到两个答案？
  - 引导：x² + 2x - 3 = 0 能分解成 (x+?)(x-?) 吗？
  - 引导：配方法的核心思想是什么？为什么要加一个数再减掉？
- **常见误解**：忘记写成标准形式再使用公式

## 一次函数
- **定义**：y = kx + b (k ≠ 0)，图像是一条直线
- **性质**：k 决定增减性（k>0递增，k<0递减），b 决定截距
- **引导线索**：
  - 引导：k 变化时直线怎么旋转？b 变化时直线怎么移动？
  - 引导：两个一次函数的图像在什么情况下相交？平行？
- **常见误解**：混淆一次函数和正比例函数的关系

## 二次函数
- **定义**：y = ax² + bx + c (a ≠ 0)，图像是抛物线
- **性质**：
  - a > 0 开口向上，a < 0 开口向下
  - 对称轴：x = -b/(2a)
  - 顶点：(-b/(2a), (4ac-b²)/(4a))
- **引导线索**：
  - 引导：a 变大时抛物线变宽还是变窄？
  - 引导：顶点坐标和对称轴有什么关系？
  - 引导：二次函数图像和x轴的交点有什么意义？
- **常见误解**：误以为a越大抛物线开口越大（实际越小）

## 因式分解
- **方法**：
  - 提公因式法：ab + ac = a(b + c)
  - 公式法：a² - b² = (a+b)(a-b)、a² ± 2ab + b² = (a±b)²
  - 十字相乘法：x² + (p+q)x + pq = (x+p)(x+q)
- **引导线索**：
  - 引导：x² - 4 = 0 除了直接开平方还能怎么解？
  - 引导：(x+1)(x-2) 展开后是什么？反过来怎么分解？
- **常见误解**：用公式时忘记判断符号

## 不等式
- **性质**：两边同时乘除负数时不等号方向要改变
- **解集**：用数轴表示
- **引导线索**：
  - 引导：和方程相比，不等式的解有什么不同？
  - 引导：为什么两边乘以负数要变号？试想一下 2 > 1，两边都乘以 -1
- **常见误解**：乘以负数时忘记改变不等号方向

## 平方根与算术平方根
- **定义**：若 x² = a，则 x 是 a 的平方根，记作 ±√a
- **算术平方根**：非负的那个平方根，记作 √a
- **引导线索**：
  - 引导：²√4 = ? 为什么不是 -2？
  - 引导：(-3)² = 9，那么 9 的平方根是什么？
  - 引导：√(-4) 有意义吗？为什么？
- **常见误解**：混淆平方根（±√a）和算术平方根（√a），误以为√a表示所有平方根
