# 初三下册

## 反比例函数
- **定义**：形如y=k/x（k≠0）的函数叫反比例函数，图像是双曲线
- **性质**：k>0时双曲线在一三象限；k<0时在二四象限；x≠0
- **引导线索**：
  - 引导：y=6/x的图像在哪些象限？k>0还是k<0？
  - 引导：反比例函数的自变量x能取0吗？为什么？
  - 引导：k的几何意义是什么（与坐标轴围成的矩形面积）？
- **常见误解**：k的正负和图像象限的关系搞错

## 反比例函数的应用
- **应用**：反比例关系在实际问题中的应用（如压力与面积、速度与时间）
- **引导线索**：
  - 引导：路程一定，速度和时间成什么关系？
  - 引导：反比例函数和一次函数的图像交点怎么求？
  - 引导：反比例函数图像上的点到坐标轴围成的矩形面积为什么相等？
- **常见误解**：自变量取值范围忽略实际意义

## 相似三角形的判定
- **判定定理**：AA、SAS（两边成比例且夹角相等）、SSS（三边成比例）
- **引导线索**：
  - 引导：AA判定中需要几个角相等？
  - 引导：SAS判定与全等中的SAS有什么异同？
  - 引导：三边对应成比例的三角形为什么一定相似？
- **常见误解**：误以为两边成比例且非夹角相等也能判定相似

## 相似三角形的性质
- **性质**：对应角相等、对应边成比例；周长比=相似比；面积比=相似比的平方
- **引导线索**：
  - 引导：两个三角形的面积比和相似比有什么关系？
  - 引导：相似比为2:1时，面积比是多少？
  - 引导：利用相似可以测量哪些无法直接测量的长度？
- **常见误解**：面积比直接等于相似比

## 相似三角形的应用
- **应用**：利用相似测高、测距（如影子测量法、镜子反射法）
- **引导线索**：
  - 引导：如何利用影子和相似三角形测量树的高度？
  - 引导：河宽怎么利用相似三角形测量？
  - 引导：位似图形和相似图形有什么关系？
- **常见误解**：比例式列错

## 位似
- **定义**：两个多边形不仅是相似图形，而且对应点连线交于一点
- **性质**：位似比等于相似比；位似中心是各对应点连线的交点
- **引导线索**：
  - 引导：位似和相似有什么区别？
  - 引导：如何以原点为位似中心将图形放大2倍？
  - 引导：位似变换下的坐标变化有什么规律？
- **常见误解**：位似和一般的相似混淆

## 锐角三角函数
- **定义**：sinA=∠A的对边/斜边；cosA=∠A的邻边/斜边；tanA=∠A的对边/邻边
- **特殊角**：30°、45°、60°的三角函数值
- **引导线索**：
  - 引导：sin30°等于多少？怎么记忆？
  - 引导：在Rt△ABC中，∠C=90°，AB=10，BC=6，sinA是多少？
  - 引导：sinA和cosA有什么关系（同角关系）？
- **常见误解**：sin、cos、tan的对应关系记反

## 解直角三角形
- **方法**：利用三角函数关系求解直角三角形的边和角
- **应用**：测高、测距、坡度、仰角俯角
- **引导线索**：
  - 引导：已知直角三角形一边一角，怎么求其他边？
  - 引导：从山脚看山顶的仰角是30°，山高100m，求山脚到山顶的水平距离。
  - 引导：坡度i和tan有什么关系？
- **常见误解**：仰角和俯角搞混

## 投影
- **定义**：平行投影（光线平行）和中心投影（光线交于一点）
- **引导线索**：
  - 引导：太阳光下的影子是什么投影？路灯下的影子呢？
  - 引导：同一时刻不同物体的影长和高度有什么关系？
  - 引导：中心投影中物体离光源越近影子越大还是越小？
- **常见误解**：平行投影和中心投影的性质混淆

## 三视图
- **定义**：主视图（正面看）、左视图（左面看）、俯视图（上面看）
- **口诀**：长对正、高平齐、宽相等
- **引导线索**：
  - 引导：从三个方向看一个长方体，分别看到什么形状？
  - 引导：主视图和左视图的高为什么是相等的？
  - 引导：俯视图中的宽和左视图中的宽有什么关系？
- **常见误解**：三视图的方位对应错误

## 立体图形的展开图
- **定义**：将立体图形的表面展开成平面图形
- **常见展开图**：正方体的11种展开图、圆柱、圆锥的展开图
- **引导线索**：
  - 引导：正方体的展开图有多少种？怎么判断一个图形能否折成正方体？
  - 引导：圆柱的展开图由什么图形组成？
  - 引导：圆锥的侧面展开图是什么形状？
- **常见误解**：展开图中面的对应关系找错

## 统计与概率综合
- **综合应用**：用统计方法收集数据，用概率分析不确定性
- **引导线索**：
  - 引导：频率和概率有什么关系？频率能代替概率吗？
  - 引导：大量重复试验中频率会趋近于什么？
  - 引导：概率为0的事件一定不发生吗？
- **常见误解**：频率和概率的概念混淆
